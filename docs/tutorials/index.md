# Tutorials

Step-by-step guides to help you get started with Astronomo.

Tutorials are **learning-oriented**. They take you through a series of steps to complete a project. They're designed to help you get started and give you hands-on experience.

## What You'll Learn

By completing these tutorials, you'll understand:

- How to navigate Gemini pages using keyboard and mouse
- How to manage your browsing history
- How to organize bookmarks into folders
- How to customize the appearance and behavior
- How to handle client certificates for authenticated sites
